# 🤖 AI 模型数学能力实测排名

> 测试题目：原创高难度数学卷（24 道跨章节、几何与代数综合、实际应用题）  
> 满分：**240 分**  
> 难度参考：远超 A-Level Further Math（平均 L3～L5 级）

## 📊 最终得分排名

| 模型 | 得分 | 正确率 | 表现亮点 / 槽点 |
| :--- | :--- | :--- | :--- |
| **DeepSeek Pro** | 204 | 85.0% | 最高分，但死磕几何、跳过部分题，专家模式“一根筋” |
| **DeepSeek Flash** | 180 | 75.0% | 第二高，写得太简略扣分，几何克星，中断一次 |
| **千问 3.7 Max** | 145 | 60.4% | 中断 4 次（输出限长），号称“数学最好”却频频卡壳 |
| **豆包** | 133 | 55.4% | 听不懂人话，第一次输出卡住，勉强完成 |
| **文心一言** | 112 | 46.7% | 全场最低，几何克星，中文切英文思考 |
| **DeepSeek 多模态** | 103 | 42.9% | 政治历史强，但数学仅 103 分，令人意外 |

## 🧪 测试背景

- **题目类型**：复数几何、解析几何、立体几何、数论组合、微分方程、实际盾构机隧道建模等  
- **题目特点**：大部分为原创或深度改编，模型无法通过训练数据“背答案”  
- **对比基准**：A-Level Further Math 常规压轴题约为 L2，本卷平均 L3～L4，最高 L5  
- **评分标准**：每道题按步骤给分，满分 240 分

## 💡 核心结论

- **DeepSeek Pro** 以 85% 的正确率稳居第一，展现了最强的数学推理泛化能力。  
- **千问 3.7 Max** 相比上一代（3.6 仅 8.8 分）进步巨大，但在原创高难度测试中仍与 DeepSeek 有明显差距，且存在多次中断、步骤冗余、主动跳过难题等问题。  
- 公开榜单（如 AIME、MATH、Arena）的高分与实测能力存在显著“注水”，用户应优先采用私有、原创、高难度的测试来评估模型真实水平。
